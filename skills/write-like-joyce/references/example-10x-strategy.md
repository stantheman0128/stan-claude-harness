【 《十倍勝，絕不單靠運氣》：從南極探險看台灣新創的生存策略 】
⠀⠀⠀
長期關注 Addin Venture 的朋友可能會發現，我們的文章常常帶著一種「矛盾感」：

一方面，我們鼓勵創業者大膽作夢、相信技術能改變世界；另一方面，我們又反覆強調務實、風險控管，以及在混沌市場中如何活得更久。

這種張力並不是我們故意為之，而是因為真正的創業現場從來不是浪漫與理性二選一，而是一場長期在「勇敢作夢」與「謹慎紀律」之間取得平衡的探險旅程。

近兩年全球資本市場劇烈擺盪，我們觀察到有些團隊在逆風下依舊穩健成長；有些則在類似條件下開始失速、甚至掉出賽道。

於是回到核心問題，讓我們回頭思考：究竟是什麼決定了企業能不能在不確定時代「活下去、甚至跑得更快」？

Jim Collins 的《十倍勝，絕不單靠運氣》提供了一個極具說服力的答案。而書中的核心案例——1911 年阿蒙森與史考特的南極探險對決，更成為理解「為何身處同樣環境，但是不同團隊卻有完全不同結果」的最佳範例。

以下，我們想從這段歷史出發，重新思考台灣新創在當前環境中的生存策略與成長方法論。
⠀⠀⠀

▐  南極探險：一場關於生存、紀律與決策的極限實驗

1911 年，兩支世界頂尖的探險隊同時出發前往南極點。挪威的阿蒙森（Roald Amundsen）與英國的史考特（Robert Scott）擁有相近的資源與技術，也都是當時最受敬重的探險家。

然而五個月後，兩隊的命運卻走向完全不同的結局：阿蒙森團隊全員平安返國；史考特全隊罹難。

這個對比極強的真實故事，成為 Jim Collins 在《十倍勝，絕不單靠運氣》的核心案例，用以回答創業界最常見的問題：同樣的環境、類似的條件，為什麼有些公司能做到市場平均的十倍，有些卻慘敗？

Collins 的研究團隊從兩萬多家公司中篩選出七家十倍勝公司，他們在動盪市場中的股票報酬率，比同業平均高出十倍。關鍵不是運氣，而是三種行為模式：狂熱紀律、建設性偏執，以及以實證為依據的創造力。
⠀⠀⠀

▐  Jim Collins 的研究方法：關鍵在「對照組」

與《從 A 到 A+》《基業長青》不同的是，本書首次將「環境變數」納入研究，探討企業在高度不確定情況下如何生存與成長。

研究採用「配對比較法」：為每一家十倍勝公司挑選同時期、同產業、起跑點類似但最終失敗的企業作為對照組。

這種設計讓結論不會淪為「從成功推敲成功」，而能真正找出導致差異的決定性因素。
⠀⠀⠀

▐  給台灣新創的策略啟示
⠀⠀⠀

1. #遠離死亡線：安全邊際比速度更重要

阿蒙森在布置補給站時，不只在主站插旗，還在前後一英里的區域內每隔半英里插上標竿，確保暴風雪中也能找到方向。他準備的食物是需求的三倍，只因為一句極具紀律的原則：

「我寧願拖著更重的雪橇，也不要餓死在冰原上。」

對台灣新創而言，這段思維格外重要。

而Stripe 的成長史就是最好的例子。

我們常以為 Stripe 是靠「7 行程式碼」走紅的技術神話，但如果從更長的時間軸來看，你會發現 Stripe 真正的競爭力不僅僅是速度和技術，還包括了企業策略：「刻意保持的安全邊際」。

Collison 兄弟在早期就有一個在矽谷非常罕見的原則：

「永遠維持比競爭對手更長的 Runway。」

2010–2012 年的支付市場百家爭鳴，許多同業把剛募到的資金押在 aggressive 擴張、快速 hiring、花大錢搶市場；Stripe 的選擇完全相反：

--投資人願意加碼，他們卻先拉長 runway
--市場狂飆時，他們不急著擴張團隊
--監管不確定時，他們保留更多緩衝，確保活得更久

這背後的精神是：不為短期市場情緒加速，而是為長期不確定性備戰

Stripe 的 API 十年故事也透露出同樣的 DNA：當產品長到一個複雜度，他們不是硬撐，而是願意停下來、重構、重新定義 API 的第一性原則。

這種「放下執著、重新打地基」的決定非常反直覺——它讓他們失去當年「7 行程式碼」的美名，卻換來未來十年的可擴張性。

本質上，這是一種極高紀律的風險管理：不追求跑最快，而是確保自己永遠不會被暴風雪打趴。

支付產業的監管環境在過去15年變化極大，許多同業在幾次重大調整時撐不住、退出市場；而 Stripe 因為保留了比別人更深的安全邊際，不只是活下來，反而在每次波動後變得更強。

這就是遠離死亡線的真正含義：不是等待幸運，而是讓自己在壞運來時仍有下一步可以走。
⠀⠀⠀

2. #二十哩行軍：紀律帶來複利

書中最具啟發性的對比之一，是兩隊面對「天氣」這個不可控變數時的處理方式：阿蒙森團隊無論晴雨都維持 15–20 哩的前進節奏；史考特團隊則是天氣好時狂衝 40 哩累垮全隊，天氣差就整天躲在帳篷抱怨運氣。

這種差異，完美呈現新創最常見的問題——過度期待「立即見效」，無法承受「延遲回報」。

我們看過太多團隊陷入同樣的循環：

新策略上線第一個月全力衝刺 → 沒看到爆發就開始沮喪 → 停滯兩週 → 換個方向再次暴衝。

這種「衝刺 → 失望 → 換策略 → 再衝」的模式，就是典型的史考特式失敗方程式。

但真正的成長往往需要時間醞釀，就像種子需要在土裡待夠久才會發芽。

我們 Addin Venture 有個 Portfolio 公司，是「二十哩行軍」最好的例子。

他們從創業第一天就建立了一套簡單但嚴格的紀律：每週追蹤接觸多少潛在客戶、銷售漏斗的轉換率、客戶反饋的關鍵詞。

創業初期的九個月，數字非常緩慢，每週開會看著那些個位數的轉換率，正常人都會懷疑這樣做有沒有意義。

但他們撐住了——每週檢討、每週微調、每週前進一點點。

現在他們即將突破 100 萬美金 ARR，而且到現在每週一早上九點的銷售檢討會仍準時進行，用同一份表格追蹤指標（只是內容隨著成長變得更完整）。

這就是「二十哩行軍」的力量——不是跑得多快，而是能跑多久。

當你建立了這種節奏，團隊會產生動能、數據會開始說話、投資人會看到可預測性。

更重要的是，看似緩慢的進步，會在時間累積下產生驚人的複利效應。
⠀⠀⠀

3. #從子彈到炮彈：精準實驗，把運氣最大化

史考特帶了三台未經測試的機動雪橇，五天內全部報廢；阿蒙森則花了一年測試各種工具，最後選擇最可靠的狗拉雪橇。

這個對比直接打臉了台灣新創常見的迷思——「進軍美國市場要準備到 100 分才能出發」。

很多團隊說：「等產品更完整」、「等找到美國合夥人」、「等募到 A 輪」……結果永遠在等。

但 Collins 的「子彈策略」告訴我們：你不需要一開始就發射炮彈，你需要的是低成本的校準。

實務上可以這樣做：

※第一發子彈： 花兩萬台幣架英文網站＋投放 1000 美金 Google 廣告，測試轉換率
※第二發子彈： 寄出 100 封 cold email，測試回覆率與興趣程度
※第三發子彈： 找當地 sales agency 合作 3 個月，測試成交機率

每一發子彈都在幫你校準方向，成本可能不到台幣 50 萬，但你已經獲得最真實的市場訊號。

當數據告訴你產品有市場、客戶願意付費、銷售流程可複製時，這才是發射炮彈的時機——在當地設立公司、聘請全職業務、投入百萬行銷預算。

我們看過太多台灣團隊，還沒射過任何一發子彈，就想直接在矽谷租辦公室、聘請年薪 15 萬美金的 BD，最後燒完資金才發現產品根本不符合當地需求。

記住：美國市場很大，但它也允許你從很小的地方開始測試。

重點不是準備得多完美才出發，而是用最小成本獲得真實回饋，再用數據決定下一步。

這是阿蒙森贏得南極競賽的秘訣，也是台灣新創征服美國市場最務實有效的方法。
⠀⠀⠀

▐  建構你的十倍勝方程式

阿蒙森說：「勝利屬於把一切安排妥當的人——人們稱之為運氣。」

史考特在凍死前寫下：「我們的運氣實在太糟了。」

同樣是運氣，有人靠準備創造，有人把失敗歸咎於它。

同樣是南極，一隊全員生還，一隊全軍殞落。

差別從來不在「誰比較幸運」，而是在於誰願意投入多少準備、紀律與實驗，去換取在極端環境中的生存空間。

如果把這套邏輯放回台灣新創，我們的南極就是美國市場。

而台灣創業者在全球創業者之中，一直有幾個特別突出的優勢：執行速度快、韌性高、技術紮實、成本效率極佳。只要搭配正確的行為模式，在美國市場一樣能做到「十倍勝」。

具體來說，就是把本文的三條原則變成日常習慣：
☆用「遠離死亡線」的思維主動拉長 runway，確保永遠有下一步可以走

☆用「二十哩行軍」建立穩定節奏，讓成長變成可預測、可複利的長期行為

☆用「子彈 → 炮彈」設計實驗，先校準方向，再放大資源、放大好運

當這三件事同時發生時，你就不是在「等待好運降臨」，而是讓自己有能力抓住美國市場給你的每一次機會。

十倍勝從來不是因為幸運，而是因為你準備得足夠充分，讓運氣願意站在你這一邊。

而這也是我們在 Addin Venture 陪跑團隊時最在意的底層方程式。
